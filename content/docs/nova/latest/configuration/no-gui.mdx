---
title: No GUI
---

When running `SpigotMC` and `PaperMC` servers run they open a GUI. By default, the `noGui` option is enabled.


```groovy tab="Groovy DSL"
runServer {
    minecraftVersion("1.21.4")
    noGui(false) // opens a gui
}
```
```kotlin tab="Kotlin DSL"
runServer {
    minecraftVersion("1.21.4")
    noGui(false) // opens a gui
}
```
