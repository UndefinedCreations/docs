---
title: Component Argument
---

Must be a valid JSON text component, returns `net.kyori.adventure.text.Component`. Example:

```java tab="Java"
Audience audience = BukkitAudiences.create(this);
new StellarCommand("send-to-self")
        .addMessageArgument("message")
        .addExecution(Player.class, context -> {
            Component component = context.getArgument("message");
            audience.sendMessage(component);
        });
```
```kotlin tab="Kotlin"
val audience = BukkitAudiences.create(this)
StellarCommand("send-to-self")
    .addMessageArgument(name = "message")
    .addExecution<Player> {
        val component = getArgument<Component>("message")
        audience.sender(sender).sendMessage(component)
    }
```
