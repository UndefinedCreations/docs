---
title: Boolean Argument
---

The `BooleanArgument` allows you to input a `boolean`. Example:

```java tab="Java"
new StellarCommand("server")
    .addBooleanArgument("boolean")
    .addExecution(Player.class, context -> {
        context.getSender().sendMessage(context.getArgument<Boolean>("boolean"));
    });
```
```kotlin tab="Kotlin"
StellarCommand("server")
    .addBooleanArgument("boolean")
    .addExecution<Player> {
        sender.sendMessage(getArgument<Boolean>("boolean"))
    }
```
